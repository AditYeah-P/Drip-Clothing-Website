Welcome! We Aditya P (RA2211003010548) and Sreesh P (RA2211003010581), have brainstormed and have come up with the idea to build a "Clothing Website" project under the faculty "Geetha K" of Computer Science department at SRM University.

Here's things we want to implement in this website:

  1. A web catalog that lists clothes from different brands.
  2. A filter-based search like that of Amazon where users can select size, price and other metrics.
  3. A comment section where users can review and rate the clothes.
  4. A like and a save feature, that will store user's favourites!
  5. Deploying the project and hosting it in a public domain!

Skills Required To Complete This Project :

  • HTML
  • CSS
  • JavaScript
  • React
  • SQL
  • Java + Springboot
